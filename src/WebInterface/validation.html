Bienvenue sur l'interface de validation manuelle d'evenements-tweeter


Vous pouvez basculer à tout moment entre le mode simple et le mode avancé de l'interface en cliquant sur le bouton "Mode avancé".

Mode simple :
Dans le mode simple, plusieurs options s'offrent à vous :
- Si un tweet contient des localisations identifiées par l'API, alors vous pouvez sélectionner la localisation de la catastrophe indiquée dans le tweet en haut à gauche.
- Si un tweet contient des localisations non identifiées par l'API, vous pouvez sélectionner "Localisation non proposée"
- Si un tweet ne contient pas d'information menant à la localisation d'une catastrophe, il vous faudra sélectionner "Non localisable"
- Enfin si un tweet ne traite pas d'une catastrophe, sélectionnez "Hors-sujet"

Mode avancé :
Dans le mode avancé vous pouvez spécifier la règle avec laquelle vous avez identifié que le tweet parlait d'une catastrophe, au sens grammatical, syntaxique ou autre.
N'hésitez pas à parcourir la liste des règles déjà présentes avant d'en rajouter une.
Vous pouvez utiliser le site displacy

Une fois que vous êtes satisfait de votre sélection, appuyez sur le bouton suivant pour passer au prochain tweet.
